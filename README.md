# Ball Runner

A 2D endless runner built in Unity, inspired by the Google Chrome Dinosaur game and Red Ball 4.

"Roll. Jump. Survive."

## About

Ball Runner strips gameplay down to its purest form: one input, infinite challenge. 
Control a rolling ball, jump over obstacles, and survive as long as possible. 
The longer you last, the higher your score.

## How to Play

- **Jump:** Spacebar or Left Mouse Click
- **Restart:** Click the Restart button after Game Over

## Features

- Single-button jump mechanic
- Endless procedurally-spawned obstacles
- Increasing difficulty over time
- Score tracking with saved high score
- Minimal, high-contrast arcade visual style

## Tech

- **Engine:** Unity (2D Core)
- **Language:** C#
- **Physics:** Unity Physics 2D
- **UI:** Unity UI + TextMeshPro

## Status

In active development — this is a class prototype project.
